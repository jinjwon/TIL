# JS 산술 연산자

## **연산자**

- 연산자에 대해서 알아봅시다.

  연산자는 프로그래밍 언어에서 특정 연산을 하도록 하는 문자입니다.

  예를 들어서, 우리가 변수와 상수를 배울 때 다음과 같은 코드를 작성했었지요?

```jsx
let value = 1; // 변수 선언
value = 2; // 대입 연산자
```

- 여기서 두번째 줄에서 사용된 `=` 문자가 바로 연산자입니다. 연산자의 종류는 굉장히 많습니다. 그 중에서 `=` 는, 대입 연산자에 해당합니다. 첫번째 줄은 새로운 변수를 선언하는 것으로서, 대입 연산자에 해당하지 않습니다.
