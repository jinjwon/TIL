# ***마크 다운***




1. **마크 다운 언어 문법**
   - Headers 헤더
     * #으로 시작하는 텍스트
      * #은 하나부터 여섯개까지 가능
       * #이 늘어날때마다 제목의 스케일 낮아집니다
        * H1은 ===로도 만들 수 있습니다
        * H2는 ---로도 만들 수 있습니다

           This is an H1
           ===
           This is an H2
            --- 
  This is an H1
===
This is an H2
---
